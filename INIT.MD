Tu es un développeur de jeux vidéo 2D en JavaScript.

Ton défi : coder un jeu de plateforme inspiré de Super Mario Bros, mais avec un personnage original nommé **Super Dario Brosse**, pour éviter les problèmes de droits d’auteur.

Contraintes importantes :
- Le jeu doit être codé en **HTML + CSS + JavaScript** uniquement.
- Pas de framework externe.
- Pas d’assets ni de sprites provenant de Nintendo : tout doit être **original** (formes simples, couleurs unies, etc.).
- Idéalement, tout tient dans **un seul fichier HTML** avec les balises `<style>` et `<script>` intégrées.
- Le joueur contrôle **Dario** avec les touches :
  - Flèche gauche : déplacement à gauche
  - Flèche droite : déplacement à droite
  - Espace ou flèche haut : saut
- Il doit y avoir :
  - un “sol”
  - quelques plateformes
  - au moins un ennemi simple qui se déplace de gauche à droite
  - une condition de défaite si Dario tombe dans le vide
  - une condition de victoire (par exemple atteindre un drapeau à droite)

Objectif :
- Le code doit être **directement copiable/collable** dans un fichier `index.html` et jouable dans un navigateur moderne.
- Ajoute quelques commentaires dans le code pour expliquer les parties principales (création du joueur, gestion des collisions, gravité, etc.).

Commence par donner directement le code HTML complet (avec `<html>`, `<head>`, `<body>`, `<style>`, `<script>`).
